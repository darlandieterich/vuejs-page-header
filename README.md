# VueJs Page Header
This is a component page header for VueJs

## How import?
```
import PageHeader from "vuejs-page-header"
```

## Properties
### pagename
```
type: String
e.g.: pagename="The Name"
```

### styles
```
type: Object
e.g.: :styles="{'color': '#f6f6f6'}"
```

### bgColor
```
type: String
e.g.: bgColor="#cdd1ce"
```
### fgColor
```
type: String
e.g.: fgColor="blue"
```
### height
```
type: String
e.g.: height="35px"
```
### fontSyze
```
type: String
e.g.: fontSyze="18px"
```

### fontFamily
```
type: String
e.g.: fontFamily="Tahoma"
```
### icon
```
type: String
options: back1, back2
e.g.: icon="back2"
```

## Event
### @clickBack
```
Called when click on the back button
```
